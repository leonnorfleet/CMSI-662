
## Project Structure
---

stack_c/: C23 implementation
stack_cpp/: C++23 implementation
stack_java/: Java implementation

---

## Compilation and Execution

### 1. C Implementation (C23)
```powershell
cd stack_c
gcc -std=c23 test.c stack.c && ./a
```

### 2. C++ Implementation (C++23)
```powershell
cd stack_cpp
g++ -std=c++23 test.cpp stack.cpp && ./a
```

### 3. Java Implementation (Version 26)
```powershell
cd stack_java
javac Test.java Stack.java && java -ea Test
```
