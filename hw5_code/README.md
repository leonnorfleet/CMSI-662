# Project Overview

This project consists of two main components used to simulate and test CSRF vulnerabilities.

## Directories

### `evil/`
This directory contains the code used to simulate CSRF attacks against the target web server.
- **Command:** `npm run dev`

### `js/`
This directory contains the core web server application code.
- **Run Server:** `npm run dev`
- **Run Unit Tests:** `npm run test`

---
*Note: Ensure both environments are configured before running.*
